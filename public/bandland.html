TEMPLE RUN

Badland adalah game action side-scrolling dengan gaya endless runner yang menampilkan dunia gelap penuh jebakan dan rintangan. Pemain mengendalikan makhluk kecil melalui hutan misterius sambil menghindari perangkap dan hambatan. Game ini menekankan refleks cepat, koordinasi, dan kemampuan mengendalikan karakter di udara.

Fitur Utama

  • Gameplay side-scrolling dan endless runner.
  • Visual artistik dengan efek pencahayaan dramatis.
  • Banyak level dengan rintangan unik dan mekanik menarik.
  • Cocok untuk melatih refleks dan ketepatan kontrol.

Cara Main Singkat

  1. Sentuh layar untuk membuat karakter terbang/melayang.
  2. Hindari rintangan dan jebakan sepanjang level.
  3. Kumpulkan item untuk mendapatkan poin tambahan.
  4. Bertahan sejauh mungkin untuk mencapai skor tertinggi.
