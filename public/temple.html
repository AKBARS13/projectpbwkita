TEMPLE RUN

Temple Run adalah game endless runner di mana pemain berlari tanpa henti sambil menghindari berbagai rintangan dan kejaran monster. Pemain harus berlari, melompat, bergeser, dan berbelok dengan cepat agar tidak tertangkap atau terjatuh. Game ini menuntut refleks cepat dan konsentrasi tinggi untuk mendapatkan skor terbaik.

Fitur Utama

  • Gameplay lari tanpa akhir (endless runner).
  • Kontrol sederhana dengan gerakan geser.
  • Beragam rintangan dan jalur berliku.
  • Cocok untuk melatih refleks dan fokus.

Cara Main Singkat

  1. Geser layar untuk belok, lompat, dan berguling.
  2. Miringkan perangkat untuk mengumpulkan koin.
  3. Hindari jurang, api, dan rintangan lainnya.
  4. Bertahan selama mungkin untuk skor tertinggi.
